debthin
lean debian & ubuntu package indexes · debthin.org

What

A minimal Debian or Ubuntu container image is around 90MB. The first thing you do is apt update - and apt downloads another 90-216MB of index files describing tens of thousands of packages, the overwhelming majority of which will never be installed on a headless server. They sit in /var/lib/apt/lists, consuming space and bandwidth on every container, on every host, on every update cycle, indefinitely.

debthin serves curated Packages.gz indexes filtered to server-relevant packages across all components. Desktop, GUI, games, fonts and hardware-specific packages are excluded. The result is an index 60-96% smaller than upstream depending on suite and component. Everything else - the actual .deb files - redirects transparently to the official mirrors. Nothing is rehosted.

The list is rebuilt daily from upstream Packages files, ranked by popcon install counts. Packages not in the index can still be installed by name, or added permanently via pull request.

TL;DR

On a bare container with only curl:

bootstrap 
curl -fsSL http://debthin.org/debthin-keyring-binary.gpg \
  -o /etc/apt/trusted.gpg.d/debthin.gpg

printf "deb http://deb.debthin.org/debian trixie headless\n\
deb http://security.debian.org/debian-security trixie-security main\n" \
  > /etc/apt/sources.list

apt-get update

No gpg, no apt-transport-https, no ca-certificates required.

Configuration

Install the key

Without gpg - works on bare containers:

binary key (recommended) 
curl -fsSL http://debthin.org/debthin-keyring-binary.gpg \
  -o /etc/apt/trusted.gpg.d/debthin.gpg

With gpg:

armored key 
curl -fsSL http://debthin.org/debthin-keyring.gpg \
  | gpg --dearmor \
  | tee /etc/apt/trusted.gpg.d/debthin.gpg > /dev/null

Sources

DEB822 format:

/etc/apt/sources.list.d/debthin.sources 
Types: deb
URIs: http://debthin.org
Suites: trixie trixie-updates
Components: main contrib non-free non-free-firmware
Signed-By: /etc/apt/trusted.gpg.d/debthin.gpg

# Security goes direct - keep it independent
Types: deb
URIs: http://security.debian.org/debian-security
Suites: trixie-security
Components: main

Classic sources.list format:

/etc/apt/sources.list 
deb http://debthin.org trixie main contrib non-free non-free-firmware
deb http://debthin.org trixie-updates main contrib non-free non-free-firmware
deb http://security.debian.org/debian-security trixie-security main

apt optimisation

apt fetches package indexes uncompressed by default. This config tells apt to request .gz indexes and keep them compressed on disk - cutting index storage from ~80MB to ~20MB. Translation files add another ~30MB for languages you'll never need on a headless server. This works independently of debthin and is worthwhile on any server.

/etc/apt/apt.conf.d/99thin 
// Strip translation files and keep indexes compressed
Acquire::Languages "none";
Acquire::GzipIndexes "true";
Acquire::CompressionTypes::Order:: "gz";

Usage Examples

Dockerfile

Dockerfile 
FROM debian:trixie-slim

RUN curl -fsSL http://debthin.org/debthin-keyring-binary.gpg \
      -o /etc/apt/trusted.gpg.d/debthin.gpg \
 && printf "deb http://deb.debthin.org/debian trixie main contrib non-free non-free-firmware\ndeb http://deb.debthin.org/debian trixie-updates main contrib non-free non-free-firmware\ndeb http://security.debian.org/debian-security trixie-security main\n" \
      > /etc/apt/sources.list \
 && apt-get update \
 && apt-get install -y --no-install-recommends \
      your-package-here \
 && rm -rf /var/lib/apt/lists/*

distrobuilder

image.yaml (distrobuilder) 
files:
  - path: /etc/apt/trusted.gpg.d/debthin.gpg
    generator: download
    url: http://debthin.org/debthin-keyring-binary.gpg

packages:
  repositories:
    - name: debthin
      url: http://deb.debthin.org/debian
      suite: trixie
      components: [main, contrib, non-free, non-free-firmware]
    - name: trixie-updates
      url: http://deb.debthin.org/debian
      suite: trixie-updates
      components: [main, contrib, non-free, non-free-firmware]
    - name: security
      url: http://security.debian.org/debian-security
      suite: trixie-security
      components: [main]

apt:

      -o /etc/apt/trusted.gpg.d/debthin.gpg
  - apt-get update

Adding Packages

Packages not in the curated index are still installable - apt will resolve and fetch them from the official mirrors transparently via 301 redirect. The index just won't list them for apt search or apt-cache.

To add a package to the index permanently, open a pull request against github.com/debthin/debthin and add the package name to required_packages/debian.txt. The index rebuilds daily; merged additions appear within 24 hours.
